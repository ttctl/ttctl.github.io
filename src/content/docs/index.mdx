## Quick install

```sh
npm install -g ttctl
```

Or run directly with `npx`:

```sh
npx ttctl --help
```

The package is published to [npm as `ttctl`](https://www.npmjs.com/package/ttctl).

## Quick start

<Steps>

1. **Install the CLI**

    ```sh
    npm install -g ttctl
    ```

2. **Create `.ttctl.yaml`** in your home directory (or in the working directory where you'll run TTCtl). The recommended form references a 1Password item that holds your Toptal credentials:

    ```yaml
    # ~/.ttctl.yaml
    auth: "op://Personal/ttctl"
    ```

    The 1Password item must have `username` and `password` fields populated. TTCtl uses the [1Password CLI](https://developer.1password.com/docs/cli/) (`op`) to resolve the reference, with biometric prompt where the Desktop App integration is enabled.

3. **Verify the install**

    ```sh
    ttctl --help
    ```

4. **Read the docs**

    - [Installation →](/installation/) — npm and MCP-client integration
    - [Configuration →](/configuration/) — `.ttctl.yaml` format and discovery
    - [Commands →](/commands/) — CLI reference
    - [MCP Tools →](/mcp-tools/) — MCP server tools

</Steps>

## How it works

TTCtl drives Toptal Talent's own web and mobile interfaces using your session, exclusively for personal use against your own profile. The GraphQL surfaces it consumes (mobile gateway, talent profile portal, scheduler) are the same ones the official Toptal Talent web and mobile apps use; TTCtl synthesizes a local schema from observed traffic and persisted-query introspection.

For the surfaces that sit behind Cloudflare bot-management, TTCtl uses TLS-fingerprint impersonation ([`node-wreq`](https://github.com/alexey-pelykh/node-wreq)) so that JA3/JA4 and ECH validation pass like a real browser. Cookies are stored locally in Mozilla format with `0600` permissions; `cf_clearance` is JA3+IP bound and prompts for refresh from a live browser session when invalidated.

For source code, issues, and contribution guidelines, see the [main repository](https://github.com/alexey-pelykh/ttctl).
